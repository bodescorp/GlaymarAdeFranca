# Next.js 14 App Router: o que aprendi migrando 3 projetos

A migracao para o App Router mudou a forma de estruturar rotas, dados e componentes.

## Aprendizados
- Server Components trazem ganhos, mas exigem disciplina.
- Streaming melhora UX, mas precisa de limites claros.
- Cache e revalidacao pedem boas estrategias por pagina.

## Fechamento
A nova arquitetura funciona muito bem, desde que o time tenha clareza dos boundaries entre client e server.
