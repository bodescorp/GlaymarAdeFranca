# n8n + IA: automatizando workflows internos sem codigo proprietario

Um relato pratico de como usei n8n e modelos locais via Ollama para automatizar processos internos.

## Casos de uso
- Triagem de emails com classificacao automatica.
- Geracao de relatorios recorrentes.
- Onboarding com checklist inteligente.

## Conclusao
A combinacao n8n + IA local trouxe ganhos reais sem dependencias de SaaS externo.
