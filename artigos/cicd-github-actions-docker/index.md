# CI/CD com GitHub Actions e Docker: setup completo para startups

Aqui esta o pipeline que uso para deploy automatico de multiplas plataformas com baixo custo.

## Stack
- GitHub Actions com workflows reutilizaveis.
- Docker multi-stage para builds leves.
- Nginx para servir artifacts estaticos.

## Beneficios
Menos retrabalho manual, releases previsiveis e ambientes replicaveis.
